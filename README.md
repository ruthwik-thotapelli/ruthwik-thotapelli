### 01 — TalentBridge
**Secure Enterprise Hiring Platform** · [Live →](https://talent-bridge-secure-enterprise-ful.vercel.app)

Hiring pipelines fail when access isn't isolated and screening isn't automated. TalentBridge fixes both — a centralized recruitment system with role-locked permissions and automated candidate evaluation.

- **RBAC + JWT** — isolated Admin / Recruiter / Candidate access with zero permission leakage
- **Modular REST architecture** — Node.js/Express services built for extensibility, not just delivery
- **ATS-inspired screening engine** — automates shortlisting, removing manual triage bottlenecks
- **Optimized MySQL schema** — tuned for consistency and high-throughput retrieval

`React.js` `Node.js` `Express.js` `MySQL` `JWT` `REST APIs`

### 02 — CloudForge
**Production-Grade AWS CI/CD Infrastructure**

Manual deploys don't scale and configuration drift kills reliability. CloudForge is a fully automated, cloud-native pipeline that removes both from the equation.

- **AWS ECS Fargate** — serverless container orchestration, no infra management overhead
- **GitHub Actions → Docker → ECR → ECS** — one push, one repeatable, reliable release
- **ALB + auto-scaling + health checks** — built for resilience under variable load
- **CloudWatch + SNS observability** — real-time monitoring and proactive incident alerting

`AWS (ECS · ECR · ALB · CloudWatch · SNS · IAM)` `Docker` `GitHub Actions` `CI/CD`

</td>
</tr>

<td width="100%">

### 03 — Shared Expense Manager
**Financial Reconciliation Platform** · [Live →](https://shared-expense-manager-xi.vercel.app/)

Group expenses break down without transparency and automated auditing. This platform reconciles, simplifies, and visualizes shared finances end-to-end.

- **CSV anomaly detection engine** — auto-flags duplicates, invalid records, currency mismatches
- **Debt-simplification algorithms** — computes optimal "who owes whom" settlements
- **Dynamic group state handling** — preserves historical balance integrity through membership changes
- **Live analytics dashboards** — React + Recharts visualizations of spend and settlement trends

`React.js` `Vite` `Tailwind CSS` `Recharts` `Node.js` `Express.js` `SQLite`
